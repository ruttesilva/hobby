Conhecendo as notas

As notas musicais, usadas no sistema musical ocidental, são compostas por doze sons diferentes. Temos sete sons principais denominados notas naturais, que consequentemente geram outros cinco sons (acidentes).

Existem sete notas naturais, são elas: DÓ, RÉ, MI, FÁ, SOL, LÁ e SI.

O que é um tom? E um semitom?

Para termos uma relação concreta entre os sons é necessário o uso de uma medida padrão entre eles. Esta unidade é chamada de Tom, que pode ser designada da seguinte maneira:

Tom é a unidade de medida que serve para calcular o espaço existente entre dois ou mais sons. O Tom ainda pode ser dividido em duas partes, sendo chamado assim de Semitom.

Semitom é o menor intervalo existente entre duas notas na música ocidental. O que é o mesmo que tocar duas teclas consecutivas no teclado.
